Detailed Description

Structure defining a state transition.

A state transition is defined by it's

  • source pySm_stateTransitionType::sourceState and
  • destination state pySm_stateTransitionType::destinationState ,
  • it's transition condition pySm_StateTransitionType::transitionTest (given by the transition test function) ,
  • the transitions priority pySm_stateTransitionType::transitionPriority
  • and actions to be performed, if an transition has been triggered, given by the transition action function pySm_stateTransitionType::transitionAction
