Detailed Description

Structure defining a state.

A state is defined by it's

  • entry function PySm_StateType::onEntryState
  • during (state main function) PySm_StateType::onState
  • exit function PySm_StateType::onExitState. Each of these functions can be generated as NULL_PTR, if not needed
